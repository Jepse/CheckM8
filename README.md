## Getting Started

To get a local copy up and running for now, follow these steps.

### Prerequisites

  ```sh
  NodeJs/npm
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Jepse/Checkmate.git
   ```
2. Install NPM packages in the `frontend` directory
   ```sh
   cd frontend
   npm install
   ```
To run the application:

```sh
npm start
```
